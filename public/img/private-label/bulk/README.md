# Bulk Packaging Images

This directory contains the product images for the bulk packaging options page.

## Required Images

Please upload the following image files to this directory:

1. **bulk-5kg-bucket.jpg** - 5 kg bucket container image
2. **bulk-5kg-jerrycan.jpg** - 5 kg jerry can container image  
3. **bulk-1kg-flask.jpg** - 1 kg flask container image
4. **bulk-500g-flask.jpg** - 500 g flask container image
5. **bulk-1kg-bucket.jpg** - 1 kg bucket container image (smaller size)

## Image Specifications

- Format: JPG
- Recommended size: 1280x1280 pixels or similar square aspect ratio
- Background: Clean, professional (gray or white recommended)
- Object: Centered, well-lit product shots

## Directory Path

The full path to this directory is:
```
/public/img/private-label/bulk/
```

Or in your local file system:
```
public/img/private-label/bulk/
```

Once the images are uploaded, they will automatically display on the bulk packaging options page at `/private-label/bulk`.
